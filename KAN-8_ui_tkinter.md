# KAN-8: UI на Tkinter — главное окно, настройки, индикатор

> **Jira:** KAN-8 · Проект: Разработка приложения для управления впн · Статус: К выполнению

---

Реализовать GUI на Tkinter (`awg_gui/ui.py`) и Controller (`awg_gui/controller.py`).

## Главное окно

- Заголовок: «Управление AmneziaWG VPN».
- Список конфигураций (Treeview): «Страна / Город / Сервер», группировка по стране.
- Кнопки: «Подключить», «Переключить», «Отключить».
- Статус-бар внизу: цветной индикатор (серый/жёлтый/зелёный/красный) + текст «Отключено» / «Подключение…» / «Подключено: Germany · Berlin S7» / «Ошибка: …».
- Индикатор интернета: иконка/цвет, обновляется по callback от `network_checker`.

## Окно настроек

- Путь к каталогу конфигураций.
- Чекбокс «Запускать VPN при загрузке системы» (через `vpn_service.enable_autostart`).
- Чекбокс «Запускать приложение при входе в систему» (через модуль `autostart`).
- Кнопка «Восстановить исходный wg0.conf».

## Threading

- Все долгие операции (подключение, переключение) выполняются в `threading.Thread`.
- Результат возвращается в UI через `queue.Queue` + `root.after(100, drain_queue)`.

## Локализация

- Все строки в `awg_gui/i18n.py` или dict — для будущего перевода.

## Также в этой подзадаче

- `awg_gui/autostart.py` — управление `~/.config/autostart/awg-gui.desktop` (создать/удалить файл, проверить наличие).

## Acceptance

- Окно открывается, список заполнен.
- Подключение/переключение/отключение проходит без блокировки UI.
- Состояние интернета корректно отображается в реальном времени.
- Включение/выключение автозапуска приложения корректно создаёт/удаляет desktop-файл и переживает перезагрузку.
