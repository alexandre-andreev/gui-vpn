Реализовать модуль `awg_gui/config_manager.py` для работы со списком `.conf`.

## Что нужно

- Сканирование каталога (по умолчанию `/home/alexandre/Загрузки/vpn-awg/`), исключая `wg0.conf`.
- Парсинг имени файла → `ConfigEntry(country, city, server_id, filename, path)`. Например, `AustriaGrazS1.conf` → `Country: Austria, City: Graz, Server: S1`. Учитывать случаи `USAUtah.conf`, `HongKongCentralDistrictS1.conf`, `CanadaMontreal.conf`.
- Чтение содержимого `.conf` (для предпросмотра в UI).
- Сортировка по стране, потом по городу.

## Тесты (pytest)

- Парсинг каждого имени из реального списка из ТЗ (24 файла) — параметризованный тест.
- Корректная обработка имён без `S{N}`-суффикса.
- Каталог не существует → понятная ошибка.
- Пустой каталог → пустой список.
