# KAN-10: Документация — README, runbook, инструкция по откату

> **Jira:** KAN-10 · Проект: Разработка приложения для управления впн · Статус: К выполнению

---

Написать пользовательскую и операционную документацию.

## Состав

1. **README.md** — описание проекта, требования, установка, быстрый старт, скриншоты.

2. **docs/runbook.md** — операционные инструкции:
   - Как запустить приложение.
   - Как проверить статус туннеля (`wg show`, `systemctl status awg-quick@wg0`).
   - Что делать при ошибках `pkexec` / отсутствии прав.
   - Где искать логи (`~/.local/share/awg-gui/awg-gui.log`).

3. **docs/rollback-to-bash.md** — инструкция по откату:
   - Как использовать существующие bash-скрипты при неработоспособности приложения.
   - Как восстановить исходный `wg0.conf` (через настройки или вручную из `/etc/amnezia/amneziawg/wg0.conf.original`).

4. **docs/install.md** — пошаговая установка с PolicyKit-политикой.

5. **CHANGELOG.md** — история версий.

## Acceptance

- README объясняет, что делает программа, за ≤30 секунд чтения.
- Runbook покрывает 5 топ-сценариев диагностики.
- Инструкция по откату проверена: при удалённом приложении пользователь восстанавливает исходный VPN своими bash-скриптами.
