# KAN-7: network_checker — фоновая проверка интернета

> **Jira:** KAN-7 · Проект: Разработка приложения для управления впн · Статус: К выполнению

---

Реализовать модуль `awg_gui/network_checker.py`.

## Что нужно

- Класс `NetworkChecker` с `start()`, `stop()`, callback `on_status_change(connected: bool)`.
- Фоновый поток с настраиваемым интервалом (по умолчанию 7 сек).
- Проверка через `requests.get("https://1.1.1.1", timeout=3)` или `requests.get("https://www.google.com/generate_204")`.
- Запасной вариант — `socket` пинг по `8.8.8.8:53`.
- Состояние `connected` меняется только если 2 цикла подряд показали один результат (debounce, чтобы не дёргать UI).

## Тесты

- Мок `requests.get` — успех/таймаут/ошибка.
- Тест debounce: один сбой не меняет статус.

## Acceptance

- Отключение сетевого кабеля → callback срабатывает в течение 15 сек.
- Восстановление сети → callback возвращает `connected=True` в течение 15 сек.
- Поток корректно останавливается при `stop()`.
